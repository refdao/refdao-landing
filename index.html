Let's get UIs paid!

Helping web3 user interfaces earn revenue.

Running a web3 UI in most cases is NOT profitable

UIs are often an afterthought, but are key to mass adoption and require a lot of work:

  • UIs need to be built.
  • UIs need to be user-friendly.
  • UIs need to be maintained.
  • UIs have running costs (domain, servers, maintenance).
  • UIs are not risk-free, regulation may attack them.

Referral rewards for UIs

RefDAO operates a set of referral smart contracts that help UI get referral rewards.

The referral smart contract receives a transaction from a UI and directs it towards the chosen DeFi protocol, but chips away a small referral reward/fee.
